# Java Algorithms Toolkit

A collection of algorithmic problem-solving implementations developed in Java.

## Features
- Command-line based programs
- Efficient input/output handling
- Implementation of core algorithms and data structures

## Implemented Algorithms
- Alphabet position finder
- Square root calculator
- Bronze medalist finder
- Binary Search Tree (insert, traversal)
- Grid-based resource optimisation

## Technologies
- Java
- Standard Input/Output

## Overview
This project showcases algorithmic problem-solving skills using Java. Each program is designed to solve a specific computational problem efficiently.
